Getting Started: C++ in Visual Studio Code

In this quick guide, we will install C++ to use with Visual Studio Code in Windows 10.

Pre-Requisites

  • Latest version of Visual Studio Code installed

Installing C++

As the first step, we will get C++ installed. We will be using free, opensource GNU compiler made available via the MinGW-w64 (Minimalist GNU for Windows).

More information on MinGW is available at mingw.org

The installer can directly be downloaded from Sourceforge by visiting:

https://sourceforge.net/projects/mingw-w64/

Once downloaded, open the installer and set options as below:

You could set a desired path for installation, however, I would recommend to use the default option as below:

Click Next to start the installation. It may take a few minutes to complete.

Setting Environment Variables

As the MinGW installer wont add the binary path to Environment Variable list (%PATH%), we need to add it manually.

Using search option available at Windows taskbar, search for “Environment Variables” and choose Edit the system environment variables:

Note: If you are in a school/ corporate network, this option may not be available. In that case, choose Change my environment variables

And, choose: Environment Variables > Path > Edit

Click on New and add the path to your MinGW installation path. If you used the default path during the installation, it may be similar to:

C:\Program Files\mingw-w64\x86_64-8.1.0-win32-seh-rt_v6-rev0\mingw64\bin

Confirm the installation

Your installation steps are now complete and we can verify it by using your favorite commandline utility:

g++ --version
gcc --version

You should now be able to view messages similar to below:

Your code now would have executed successfully and you can view the result at Terminal window.
